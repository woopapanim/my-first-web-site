HTML이란 무엇인가?

Hypertext markup language(HTML) is the standard markup langage for creating web pages and web apoplications. Webrowsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the documents.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects , such ad interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantucs for text such as headings, paragraphs, lusts, links, quotes and other items. HTML elements are delineates by tags, written using angle brackers.
